Question 4

What is the difference between a star schema and a snowflake schema?

1 / 1 point
Correct

A star schema is highly denormalized so that the fact table joins directly to dimensions; a snowflake schema normalizes some dimensions into multiple tables such as DimProduct, DimProductSubcategory, and DimProductCategory.
